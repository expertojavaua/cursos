Estructura de directorios del CD (y de la web)

/doc: documentación
/soft: programas
/plj: web
/plj/index.htm: fichero principal de la web
/plj/restringido/apuntes: directorio con los apuntes
/plj/restringido/ejercicios: directorio con los ejercicios

Todas las referencias a documentación o a programas en recursos desde las páginas de apuntes o de ejercicios debe hacerse de forma relativa:

../../../doc/<fichero>
../../../soft/<fichero>

Ejercicios módulo 1:

  • Sesión 1: Compilación y ejecución de programas Java, CLASSPATH, paquetes, estructuras de control. Ejemplos: hola mundo, botón, hola mundo con paquetes, el juego de adivinar. Ficheros jar.
  • Sesión 2: Eclipse, API de Java, javaDoc. 
  • Sesión 3: Métodos y objetos. Herencia. Modificadores static y private.
  • Sesión 4: Interfaces, arrays, colecciones.
  • Sesión 5: Programa competo.

Ejercicios módulo 2:

  • Sesión 6: Excepciones en Java. Definición de excepciones, captura, lanzamiento, y construcción de excepciones propias.
  • Sesión 7: Hilos en Java. Concepto de hilo, formas de definir un hilo en Java, prioridades, sincronización y grupos de hilos.
  • Sesión 8: Entrada/Salida básica en Java. Ejercicio simple de lectura de ficheros
  • Sesión 9: Conceptos avanzados de entrada/salida (lectura mediante tokens). Ejercicio de lectura y parseo de ficheros.
  • Sesión 10: Ejercicio completo de lectura de ficheros utilizando excepciones.

Ejercicios módulo 3:

  • Sesión 11: Desarrollo de aplicaciones gráficas con AWT: descripción de los componentes principales, gestores de disposición, y modelo de eventos. Ejemplo de aplicación sencilla construida paso a paso.
  • Sesión 2: Swing. Diferencias con AWT. Componentes principales de aplicaciones Swing. Conversión de aplicaciones Swing a AWT e implementación de aplicaciones más elaboradas con Swing.
  • Sesión 3: Applets. Diferencias entre applets y aplicaciones. Componentes de un applet. Conversión de aplicaciones standalone a applets, e implementación de un applet completos.
  • Sesión 4: Introducción a los gráficos en Java. Dibujar en canvas y objetos de pantalla, doble buffer y otras técnicas. Ejemplos de aplicaciones con gráficos. Implementación de ejemplos sencillos de tratamiento de gráficos.
  • Sesión 5: Construcción de una aplicación gráfica completa, a elegir: 
    • Aplicación Swing con diferentes paneles, lectura/escritura de ficheros, tratamiento de excepciones, etc.
    • Implementación de un juego Java sencillo, haciendo uso de paneles Swing introductorios y de gestión de gráficos a pantalla completa.
