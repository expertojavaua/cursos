Presentación

La tecnología de los dispositivos móviles ha avanzado rápidamente en los últimos años, llegando a tener los dispositivos de última generación una capacidad similar a la de los ordenadores de hace unos años, con un tamaño muy reducido y por lo tanto una portabilidad mucho mayor.

Cada dispositivo cuenta con su propia interfaz de programación, por lo que las aplicaciones desarrolladas en general funcionan únicamente en un solo modelo. Sin embargo, los móviles de última generación soportan una interfaz Java estándar (plataforma Java ME, Java Micro Edition) que, al ser una plataforma Java, nos permitirá desarrollar aplicaciones para cualquier dispositivo independientemente del modelo y la marca.

Durante este año han surgido gran cantidad de proveedores que ofrecen aplicaciones para estos móviles, como por ejemplo juegos Java, y aplicaciones de uso personal. Pero además, Java ME nos permitirá utilizar el móvil como front-end de grandes aplicaciones de empresa (Java EE).
