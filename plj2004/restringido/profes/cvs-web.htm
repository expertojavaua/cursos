Cómo actualizar la web del curso PLJ

1. Resumen

La web se mantiene en un repositorio CVS situado en el servidor web (www.j2ee.ua.es). Para modificar la web (ver punto 2) tienes que hacer un checkout de la web en tu ordenador, modificar los ficheros que desees y hacer un commit. Muy importante: el commit modifica el repositorio CVS, pero no el sitio web. Para actualizar el sitio web (ver punto 3) hay que hacer ssh al servidor y hacer un checkout del repositorio en el directorio donde se encuentra la web.

Necesario:

  • Eclipse (las pantallas están cogidas de la versión 3.0, milestone 9, pero supongo que también funciona igual en la versión 2.1).
  • Editor sitios web (por ejemplo, Dreamweaver Mx)
  • El repositorio con el sitio web debe estar creado

3. Actualización del sitio web

Para actualizar el sitio web debes conectarte con ssh a www.j2ee.ua.es y entrar con usuario web (contraseña j2eej2ee). Una vez en tu directorio home, verás un enlace simbólico al sitio web plj. Debes ejecutar:

% cd plj
% cvs checkout -d . web/plj

Para hacerlo más fácil, he creado un script en el directorio home que ahorra teclear esto. Hay que llamarlo de la siguiente forma (estando en el directorio home de web):

% ./checkout plj
